# Introduction

Cerulean is an opinionated code formatter for the [Teal](https://teal-language.org/) programming language.

It enforces a consistent style automatically, so you can stop arguing about formatting and focus on the code. Pass a file or directory to `ceru` and it will rewrite your sources in place. Use `--check` in CI to reject unformatted code without modifying anything.

Cerulean works with Lua 5.1 and above, including LuaJIT.
